# CrewFlow Frontend

Modern Next.js operations console for the CrewFlow backend.

## Setup

```bash
yarn install
cp .env.example .env.local
yarn dev
```

Default app URL:

```text
http://localhost:3000
```

Default API URL:

```text
http://localhost:3002/api
```

Demo login:

```text
owner@sparkle.test / Password123!
```

## Views

- Overview
- Inbox
- Bookings
- Field jobs
- Invoices and payments
- Manager actions
